# HR Leave Management System

A complete HR Leave Automation System built using FastAPI.

## Features
- Leave request form
- Email approval workflow
- Google Sheets integration
- Manager approve/reject system
- Success, Approved & Rejected pages
- Loading spinner UI

## Tech Stack
- FastAPI
- Python
- HTML & CSS
- Google Sheets API
- Gmail SMTP

## How to Run

```bash
uvicorn main:app --reload
